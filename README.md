# Safari Zone – React Web Project

Questo progetto nasce come **esperimento personale e progetto da portfolio**, con l’obiettivo di creare una **web app interattiva ispirata al concetto della “Zona Safari”** dei giochi Pokémon, utilizzando **React** e JavaScript moderno.

## Obiettivo del progetto
L’idea è sviluppare una piccola esperienza esplorativa basata su:
- movimento su una mappa a griglia
- incontri casuali in determinate aree
- un semplice sistema di cattura
- inventario e progressione salvata localmente

Il focus principale non è il realismo grafico, ma:
- architettura dei componenti React
- gestione dello stato
- logica applicativa
- chiarezza e manutenibilità del codice

## Stato attuale
 **Il progetto è in fase iniziale**  
Le funzionalità verranno implementate progressivamente, partendo da:
- struttura base dell’app
- rendering della mappa
- gestione del movimento del giocatore

## Tecnologie previste
- React
- JavaScript (ES6+)
- CSS (layout a griglia)
- LocalStorage per il salvataggio dei dati
